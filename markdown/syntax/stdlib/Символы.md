
#Символы / Chars

##Свойства
    
###ПС / LF
Доступ: Чтение
    
    
Символ перевода строки.


  
  
###ВК / CR
Доступ: Чтение
    
    
Символ возврата каретки.


  
  
###ВТаб / VTab
Доступ: Чтение
    
    
Символ вертикальной табуляции.


  
  
###Таб / Tab
Доступ: Чтение
    
    
Символ табуляции.


  
  
###ПФ / FF
Доступ: Чтение
    
    
Символ промотки.


  
  
###НПП / Nbsp
Доступ: Чтение
    
    
Символ неразрывного пробела.
